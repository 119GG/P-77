Project-77 About Console and VSC

Visual Studio Code is an IDE(integrated development environment) app. It helps you code online/offline. VSC is really easy to set up and use.

STEP 1: Download VSC

STEP 2: Along with this, download/install the Live-Server Extension

STEP 3: Make a file on your computer, which is named after what you are planning to code.

STEP 4:Open VSC and click "open folder", then choose the folder you just created.

STEP 5: Start Coding!!!
